## Structure du dépôt

Le projet contient **4 dossiers**, chacun correspondant à un TP de CEM.  
Chaque dossier comprend :

- Plusieurs **PDF** : l'énoncé, le compte rendu du TP, ... (Le nom du binôme dans le cas échéant se trouve dans le compte rendu)
- Un `README.md` si le compte rendu n'est pas au format PDF
- Les **codes sources** utilisés pendant la séance  
- Les **images**, graphiques ou captures de mesures associées
